// COMPUTING_POWER

We used the Raspberry Pi 3A+ to meet the dimensions while still having enough compute to run an FFT onboard.

// DIMENSIONS

Height: 3.7 inches | Diameter: 3 inches | Weight: 0.55 lb — barely meeting requirements.

// POWER

4 AA batteries with a 5V regulator for 5+ hours of constant operation without external power.

// BACKGROUND

This project was made for the Georgia Tech NASA Mini Payload competition. We were one of four teams to progress to the CDR (Construction and Design Round) and receive a $100 stipend.

// APPLICATIONS

The findings have broad applications in optimizing communication networks for disaster response, aerospace and defense operations, identifying dead zones, and improving in-flight Wi-Fi and GPS systems.

// COMPETITION_CRITERIA

  • Must not exceed 0.75 lb
  • Fully self-contained, ≤3.7" height, ≤3" diameter
  • Center of gravity within 0.3" of centerline
  • No data transmission during flight
  • 5+ hours battery life
