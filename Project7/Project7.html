// 2D_FWD_KINEMATICS (FK.PY)

DOWNLOAD

Calculates end-effector (hand) position from joint angles. Used for "where is my hand?" when I know my joint angles. Implements scratch-built trigonometry (Taylor Series) and custom matrix multiplication.

  • Input: Link Lengths (L), Joint Angles (θ).
  • Algorithm: Computes end-effector position using explicitly implemented trigonometry (Taylor Series) and custom Matrix Multiplication.
  • Output: End Effector Coordinates (x, y) and Orientation (θ_total).

// 2D_INV_KINEMATICS (IK.PY)

DOWNLOAD

Solves the "Reaching" problem: Given a target (x, y), what angles do I need? Uses an iterative gradient descent approach (Jacobian-like optimization) to minimize distance to target.

Interactive: Click and Drag the canvas to move the target.

  • Input: Target Coordinates (x, y), Link Lengths.
  • Algorithm: Iterative Gradient Descent. Adjusts joint angles to minimize distance to target using a custom Jacobian-like approach.
  • Output: Required Joint Angles (q1, q2...) to reach the target position.

// 3D_FWD_KINEMATICS (3DFK.PY)

DOWNLOAD

Extends Forward Kinematics into 3D space. Handles rotation around arbitrary axes (x, y, or z) using complex 4x4 transformation matrices.

  • Input: Joint Angles, Axes of Rotation (x/y/z), Link Lengths.
  • Algorithm: Generates 4x4 Homogeneous Transformation Matrices for each joint (Rotation + Translation) and multiplies them sequentially.
  • Output: Final 4x4 Transformation Matrix containing 3D position (x, y, z) and orientation.

// 3D_INV_KINEMATICS (3DIK.PY)

DOWNLOAD

Advanced solver for 6-DOF robotic arms in 3D space. Calculates joint configurations to reach any point (x, y, z) while maintaining orientation constraints.

Interactive: Click and Drag to move target in 3D space.

  • Input: Target Position (x, y, z), Link Lengths.
  • Algorithm: Cyclic Coordinate Descent (CCD) optimization to solve for 6-DOF robotic arms in 3D space. Includes reachability checks.
  • Output: Optimal Joint Angles to reach target while maintaining constraints.
