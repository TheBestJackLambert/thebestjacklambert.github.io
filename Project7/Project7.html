What this tool does

          

            This  visualizer computes and draws the individual vectors for any number of revolute joints. Enter link lengths and joint angles (in degrees)
            to see the arm update in real time, the orientation, the end‑effector pose, and the homogeneous transform.
          


          
Features

          
    
            
  •  Any number of revolute links (set N and go)
    • 
            
  •  Real‑time canvas rendering & grid
    • 
            
  •  Reach circle and optional path trace
    • 
            
  •  Output with affector matrix
    •

Math (Planar FK)

          

            For links \(L_i\) and joint angles \( \theta_i \) (counter‑clockwise, in degrees here), the end‑effector is:
          

          
x = Σ L_i cos(Σ θ_j),   y = Σ L_i sin(Σ θ_j)
θ_total = Σ θ_i
T = [[cos θ, −sin θ, x],
     [sin θ,  cos θ, y],
     [   0,      0, 1]]
