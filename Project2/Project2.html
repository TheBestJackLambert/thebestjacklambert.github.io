Overview

Our goal during the Wi-Fi detector project was to create a wireless device that could detect if a Wi-Fi signal was nearby and tell us how strong it was using the brightness of an LED bulb.

Technologies Used

  • ✔ Germanium Diodes
  • ✔ LT Spice
  • ✔ Digital Signal Processing
  • ✔ Soldering

Key Features

  • ✔ Changing brightness based on signal strength
  • ✔ Wireless
  • ✔ Able to pick up 2.46Ghz and 5Ghz signals
