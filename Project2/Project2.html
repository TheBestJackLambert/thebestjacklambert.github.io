GTRI Internship Project

WI-FI DETECTOR

Portable RF device for detecting and analyzing Wi-Fi signal strength.

GTRI RF SENSORS PORTABLE CIRCUIT DESIGN

// MISSION_OVERVIEW

This project involved designing a hand-held detector capable of sniffing 2.4GHz Wi-Fi signals. The device provides real-time visual feedback via LED intensity corresponding to signal strength, allowing for physical area mapping of network coverage.

// TECHNOLOGIES

Germanium Diodes LT Spice Simulation Signal Processing Antenna Design

// KEY FEATURES

  • > Portable & Battery Powered
  • > Real-time Analog Feedback
  • > Dual Band Detection (2.4/5GHz)
